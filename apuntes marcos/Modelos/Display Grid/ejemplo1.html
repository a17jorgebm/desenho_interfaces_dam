Creación del Grid Layout

Para aplicar una estructura de rejilla tenemos que crear un elemento contenedor al que le aplicamos la propiedad display: grid; o display:inline-grid;

Los hijos directos de este contenedor se convertirán en elementos de rejilla (grid items).
