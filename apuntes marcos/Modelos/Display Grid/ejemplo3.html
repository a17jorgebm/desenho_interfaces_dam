Propiedad grid-gap

Con la propiedad grid-gap podemos establecer el espacio entre filas y columnas. Hay varias formas de aplicar esta propiedad:

  • grid-gap: 50px; Define un espacio de 50 píxeles entre las columas y también entre las filas.
  • grid-gap: 50px 25px; Define un espacio de 50 píxeles entre las filas y un espacio de 25 de píxeles entre las columnas.
  • grid-column-gap: 50px; Define un espacio de 50 píxeles entre las columas.
  • grid-row-gap: 75px; Define un espacio de 75 píxeles entre las filas.
